# Evidence Spec

## Evidence Capture Convention
- 경로: `.sisyphus/evidence/<task-id>/`
- 최소 파일:
  - `review-report.md`
  - `verification-log.md`

## Required Evidence Fields
- task-id
- command executed
- observed result
- pass/fail
- timestamp

## Forbidden Evidence Patterns
- “정상일 것 같음” 같은 추정 문구
- 실행 명령/출력 없는 주장
